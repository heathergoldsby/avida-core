Obtaining cmake : see www.cmake.org.

Documentation of cmake : see www.cmake.org/HTML/Documentation.html.

Running cmake (the concise way) : $ pwd

/Users/kaben/Projects/Software/Avida/consistency-tests

$ ls

CMakeLists.txt CMakeModules Doc Support Tests

$ mkdir build

$ cd build

$ cmake -D_CONSISTENCY_TEST_PRIMITIVE:BOOL=ON ..

Running cmake (the verbose way) :

$ pwd

/Users/kaben/Projects/Software/Avida/consistency-tests

$ ls

CMakeLists.txt CMakeModules Doc Support Tests

$ mkdir build

$ cd build

$ ccmake ..

Press 'h' for help; use up and down keys to navigate, 'return' to enter a field so you can change it; press 'c' to configure the makefiles, and 'g' to generate the makefiles.

Fields which are marked with an asterisk represent changed and unsatisfied variable dependencies -- just keep pressing 'c' until the asterisks are all gone.
